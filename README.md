# 🧠 Multi-Modal AI System (CNN + LSTM)

## 📌 Overview
This project is a Multi-Modal AI application that combines:
- 🖼 Convolutional Neural Network (CNN) for image classification
- 📝 Long Short-Term Memory (LSTM) for text sentiment analysis
- 🎨 Streamlit for interactive web UI

Users can upload an image or enter text and get instant predictions.

---

## 🚀 Features
- Image classification using CNN
- Sentiment analysis using LSTM
- Real-time Streamlit web app
- Easy user interface
- Supports multiple input types (image + text)

---

## 🧠 Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Streamlit
- NLP (Tokenizer + LSTM)
- CNN (Image Processing)
